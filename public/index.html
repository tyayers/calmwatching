CalmWatching.com
CalmWatching.com provides creative-commons, ad-free background content for second monitors, TVs or other screens.

The scene is automatically changed every 5 minutes, or on demand.

Touchscreen controls

Swipe left: load a new scene
Swipe right: go back to previous scene
Double tap: start or exit fullscreen

Keyboard controls

"r, arrow-right": load a new scene
"arrow-left": previous scene
"space, p": pause scene
"f": start or exit fullscreen
